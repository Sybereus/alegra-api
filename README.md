# Alegra Automation System

Sistema en Python para la carga masiva de comprobantes (Boletas/Facturas) a la API de Alegra desde Excel.

## Requisitos
- Python 3.12+
- Credenciales de API de Alegra

## Instalación
1. Clonar repositorio.
2. Crear entorno virtual: `python -m venv venv`.
3. Activar entorno: `.\venv\Scripts\activate`.
4. Instalar dependencias: `pip install -r requirements.txt`.
5. Configurar archivo `.env` con tus credenciales.
